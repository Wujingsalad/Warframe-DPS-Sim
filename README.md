# Warframe-DPS-Sim
Warframe-DPS-Sim模拟器是一款对Warframe
DPS 模拟
属性输入
玩家可在模拟器中手动输入当前武器的种类，基础属性，Mod搭配，也可以使用数据库中原有的武器数据。其中用户一旦输入了数据后，这些数据将会自动保存在模拟器内，以供后续使用。
Mod 模拟
玩家可以通过Mod模拟面板，在武器上添加或者移除Mod，计算是否超过Mod容量，也可以手动修改裂罅Mod的数据。
DPS 统计
在DPS模拟界面中，可以查看本次模拟的 DPS 统计信息，包括DPH，爆发DPS，持续DPS，歧视后DPS。
